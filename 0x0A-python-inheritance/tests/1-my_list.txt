Using MyList module

Importing the class

>>> MyList = __import__("1-my_list").MyList
>>> my_list = MyList()

Check if MyList inherits from list:

>>> issubclass(MyList, list)
True

Check if append works
>>> my_list.append(1)
>>> my_list.append(3)
>>> my_list.append(4)
>>> my_list.append(2)
>>> print(my_list)
[1, 3, 4, 2]

>>> my_list.print_sorted()
[1, 2, 3, 4]

Check with negatives
>>> a = MyList()
>>> a.append(1)
>>> a.append(-2)
>>> a.append(-5)
>>> a.append(2)
>>> print(a)
[1, -2, -5, 2]
>>> a.print_sorted()
[-5, -2, 1, 2]

Check empty list
>>> a = MyList()
>>> print(a)
[]
>>> a.print_sorted()
[]
